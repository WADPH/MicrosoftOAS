    Settings

    Manage editable application configuration safely.

    Main

    For security reasons, these values can only be configured directly in .env.

    • PORT — Server Port
    • TEAMS_OUTGOING_WEBHOOK_SECRET — Webhook provides this secret
    • TENANT_ID — From Entra ID application
    • CLIENT_ID — From Entra ID application
    • CLIENT_SECRET — From Entra ID application
    • DEFAULT_USAGE_LOCATION — For example 'EN'
    • SNIPEIT_URL — Base URL of your Snipe-IT instance
    • SNIPEIT_API_KEY — API key from Snipe-IT
    • ZAMMAD_URL — Base URL of your Zammad instance
    • ZAMMAD_API_TOKEN — API token from Zammad
